# Galaxy Defender 🚀

A modern, responsive 2D game interface designed with premium aesthetics and smooth animations.

## Features ✨
- **Dynamic Start Screen**: High-quality space-themed entrance with glassmorphism and animated starfields.
- **Glassmorphism UI**: Backdrop-filters and subtle glow effects for a high-end futuristic feel.
- **Responsive Layout**: Designed to look stunning on everything from mobile phones to high-resolution desktop monitors.
- **Fluid Typography**: Uses viewport-scaled fonts (clamp) for consistent readability.
- **Pure CSS Background**: Performance-optimized animated stars without heavy image assets.

## Built With 🛠️
- **HTML5**: Semantic structure.
- **CSS3**: Advanced animations, gradients, and flexbox/grid layouts.

## Current Project Status 🚧
The core **visual design** and **responsive UI** are fully implemented. The game features a "Start Screen" ready for gameplay logic.
