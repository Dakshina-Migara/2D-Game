# Shooter 🚀
A modern, responsive 2D arcade shooter designed with premium aesthetics and smooth gameplay logic.

## 🎮 How to Play
Take control of your stellar explorer and defend the galaxy!

### Controls
*   **Move Left**: `A` key or `Left Arrow`
*   **Move Right**: `D` key or `Right Arrow`
*   **Shoot**: `Spacebar`
*   **Pause/Resume**: `Esc` key

## ✨ Features
*   **Keyboard-Only Controls**: Optimized for desktop gameplay with responsive input handling.
*   **Adaptive Difficulty**: The game gets harder as you score! The falling speed of target orbs increases every 20 points.
*   **Clean Gameplay**: Only one target orb falls at a time to ensure a focused and challenging experience.
*   **Interactive UI**: Sleek, neon-themed Game Over and Pause screens with blur effects.
*   **Performance First**: Built with pure Vanilla JavaScript and CSS for maximum efficiency.
*   **Responsive Layout**: Designed to look stunning on any screen size.

## 🛠️ Built With
*   **HTML5**: Semantic tags and game surface structure.
*   **CSS3**: High-end glassmorphism, gradients, and subtle animations.
*   **Vanilla JavaScript**: Optimized game loops, collision detection, and state management.
